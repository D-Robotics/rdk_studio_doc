# 3.12 配置中心

![配置中心（Moss 设置）：左侧 6 区域导航（账户与安全 / AI 引擎 / 飞书 / 微信 / 设备连接 / 应用与更新），右侧是账户与安全 + AI 引擎的字段](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/rdk_studio/zh/settings-panel.png)

配置中心是 RDK Studio 全局配置的统一管理界面。所有非临时性的配置项都在这一处管理：账户、AI 模型、设备连接默认值、外观、应用与更新等。

底层使用 JSON Schema 定义配置结构，便于校验、迁移、跨版本兼容。配置文件存储在本机用户目录下，可以导出与导入实现跨机器同步。

本节是 AI 引擎、设备连接默认值、配置导入导出等功能的"主住所"。在 [2.5 接入 AI 模型](../../2-quick-start/5-configure-ai-model.md) 中提到的"详细配置"全部在本节展开。
