# 3.8.2 隐藏 SSID 与高级选项

## 手动添加隐藏 SSID

部分企业 WiFi 不广播 SSID，扫描列表中找不到。这种网络需要手动输入：

1. 点击 WiFi 配置弹窗底部的 *手动添加 SSID*
2. 输入 SSID 名称（区分大小写）
3. 输入密码
4. 选择加密类型（WPA2 / WPA3 / WEP / 开放）
5. 勾选 *隐藏网络* 复选框
6. 点击 *连接*

后续重启板端时，板会主动尝试连接已保存的隐藏网络，不需要再次手动操作。

## 加密类型选择

| 加密类型 | 适用 |
|---|---|
| WPA2 | 当前最常见的家用与办公 WiFi 标准 |
| WPA3 | 较新的标准，部分新路由器使用 |
| WEP | 已淘汰的旧标准，仅在老设备上出现 |
| 开放 | 无密码网络（如咖啡店、公共 WiFi） |

如果不确定加密类型，先尝试 WPA2，失败后再试其他类型。`nmcli` 通常会在连接时自动尝试常见的加密类型。

## 切换有线与无线优先级

板端同时接入有线网络和 WiFi 时，路由默认走有线（速度更稳定）。如果需要强制走 WiFi（例如有线网络访问受限）：

```bash
# 临时禁用有线
sudo ip route del default dev eth0
```

如需永久切换，可以在板端调整 NetworkManager 的连接优先级：

```bash
sudo nmcli connection modify "<连接名>" connection.autoconnect-priority 100
```

数值越大优先级越高。

## 切换 2.4 GHz 与 5 GHz

部分双频路由器把 2.4 GHz 与 5 GHz 设为相同 SSID。`nmcli` 通常优先连接信号更强的频段，但在信号边缘场景可能频繁切换导致不稳定。

如果遇到这种问题，建议在路由器后台为两个频段设置不同的 SSID（如 `office_5g` 和 `office_2g`），然后在 Studio 中手动选择稳定的那一个。
