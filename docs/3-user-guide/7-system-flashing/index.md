# 3.7 系统烧录

![系统烧录向导 · 第 1 步选择板型：RDK X3 / X5 / S100(P) 三张卡片，S100(P) 标注 XBURN 专用通道](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/rdk_studio/zh/08-burnning.png)

系统烧录是 RDK Studio 内的一站式烧录界面，覆盖 RDK 全系列板：RDK X3、RDK X5 用 TF 卡或 eMMC，RDK S100 用专用 xburn 工具。开发者无需为不同板型安装多个烧录工具。

[2.2 烧录系统镜像](../../2-quick-start/2-flash-system.md) 提供了"快速跑通"的最简流程。本节是烧录的完整参考，包含三种介质的详细操作、eMMC 备份与还原、烧录失败的排查。

- [3.7.2 TF 卡烧录](./2-tf-card.md)：RDK X3 与 RDK X5 的标准烧录方式
- [3.7.3 eMMC 烧录与备份](./3-emmc.md)：RDK X5 带 eMMC 版本的烧录与备份还原
