# 3.4.1 目录浏览与传输

文件管理的主界面分为三部分：左侧目录树、右侧文件列表、底部任务栏。

## 界面结构

| 区域 | 显示内容 |
|---|---|
| 左侧目录树 | 板端常用根目录与当前打开的子树 |
| 右侧文件列表 | 当前目录下的文件，含名称、大小、修改时间、权限 |
| 顶部地址栏 | 直接输入路径快速跳转，例如 `/opt/tros/humble/lib/` |
| 顶部面包屑 | 当前路径的每一段都可点击跳转 |
| 底部任务栏 | 进行中的上传/下载任务，含速度与剩余时间 |

## 上传文件

| 方式 | 操作 |
|---|---|
| 拖拽上传 | 把文件或文件夹从 PC 直接拖到 Studio 窗口 |
| 上传按钮 | 顶部 *上传* 按钮，弹出文件选择器 |
| 批量上传 | 选择多个文件或整个文件夹，一次性上传 |

上传时底部任务栏显示进度。上传到敏感路径（如 `/sys`、`/proc`）受访问控制限制，详见 [3.4.3 路径访问控制](./3-path-access-control.md)。

## 下载文件

| 方式 | 操作 |
|---|---|
| 下载按钮 | 文件名右侧的下载图标，下载到 PC 默认下载目录 |
| 批量下载 | 勾选多个文件，右键 *下载选中*，打包为 zip 后下载 |
| 拖拽下载 | 在 macOS、Linux 上支持拖拽到桌面或资源管理器 |

## 右键菜单

在文件或目录上右键弹出操作菜单：

| 操作 | 作用 |
|---|---|
| 新建文件 | 在当前目录创建新的空文件 |
| 新建目录 | 在当前目录创建新的子目录 |
| 重命名 | 修改文件或目录名（也可用 F2 快捷键） |
| 删除 | 删除选中项（也可用 Delete 键） |
| 复制完整路径 | 把绝对路径复制到剪贴板 |
| 在此打开终端 | 打开 *远程终端* 新标签，自动 cd 到该目录 |

## 文件名搜索

顶部搜索框只搜索当前目录的文件名（不递归子目录），避免在大目录中卡死。如果需要递归搜索整个文件系统，建议在 AI Dock 中描述需求："在 `/opt/tros` 下找所有 .hbm 文件"，Agent 会调用 `find` 命令完成。

## 常用路径速查

| 路径 | 用途 |
|---|---|
| `/userdata/` | RDK 推荐的用户可写区，刷新系统不丢失 |
| `/tmp/` | 临时文件，重启清空 |
| `/opt/tros/humble/` | TROS 安装目录（lib、share、include） |
| `/home/<user>/` | 用户家目录 |
| `/var/log/` | 系统日志 |
| `/etc/` | 系统配置 |
