# 3.4.2 在线编辑

文件管理内置 Monaco Editor（VS Code 同款编辑器组件），可直接在 Studio 内编辑板端文件而无需下载到 PC。这一能力适用于改单个配置文件或快速查看代码——多文件、项目级开发请使用 [3.5 远程 IDE](../5-remote-ide/index.md)。

## 编辑流程

| 步骤 | 操作 |
|---|---|
| 1 | 在文件管理中双击目标文件 |
| 2 | Monaco Editor 在右侧或新窗口打开 |
| 3 | 编辑内容，编辑器实时显示语法高亮 |
| 4 | Ctrl + S 保存，自动通过 SFTP 写回板端 |

保存成功后会有提示。如果保存失败（例如权限不足），编辑器会显示错误信息。

## 支持的文件类型

Monaco Editor 默认支持以下常见格式的语法高亮：

| 类别 | 扩展名 |
|---|---|
| 配置文件 | `.yaml`、`.yml`、`.toml`、`.ini`、`.json`、`.conf` |
| 脚本 | `.sh`、`.py`、`.js`、`.ts` |
| 代码 | `.cpp`、`.c`、`.h`、`.hpp`、`.go`、`.rs` |
| 标记语言 | `.md`、`.xml`、`.html` |
| 纯文本 | `.txt`、`.log` |

未识别的扩展名会以纯文本方式显示，仍可编辑保存。

## 编辑器能力

| 功能 | 支持 |
|---|---|
| 语法高亮 | 是 |
| 行号 | 是 |
| 代码折叠 | 是 |
| 查找替换 | Ctrl + F / Ctrl + H |
| 多光标编辑 | Alt + 点击 |
| 自动保存 | 否（需要手动 Ctrl + S） |
| 撤销重做 | Ctrl + Z / Ctrl + Y |

## 大文件警告

当打开的文件超过 5 MB 时，编辑器会弹出警告，原因：

- 浏览器内编辑大文件性能较差，可能卡顿
- 通过 SFTP 保存大文件耗时较长
- 大文件通常是日志、二进制等不适合在线编辑的内容

建议：

- 日志文件先用远程终端 `head -1000 large.log > /tmp/sample.log` 截取后再查看
- 大型代码项目用 [3.5 远程 IDE](../5-remote-ide/index.md) 编辑（基于 code-server，文件直接在板端处理）
- 二进制文件不要尝试在编辑器中打开，会显示乱码
