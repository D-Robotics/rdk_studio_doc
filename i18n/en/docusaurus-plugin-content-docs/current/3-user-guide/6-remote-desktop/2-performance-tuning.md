# 3.6.2  Pop-out Window Mode

The desktop client can detach the remote desktop into an independent floating window:

- The main Studio window can continue displaying other tabs.
- The floating window supports always-on-top, fullscreen, and secondary monitor placement.
- Multiple boards' floating windows can be popped out simultaneously for side-by-side observation.

**How to use**: Click the *Pop-out Window* button in the top-right corner of the remote desktop tab’s toolbar.
