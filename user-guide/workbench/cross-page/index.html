3.1.2 跨页面联动

工作台把"读数据 → 动手处理"两件事放在同一屏：指标条给出状态、下方按钮直达对应的功能页，避免开发者在多个 tab 之间反复切换。

指标下方的 4 个主 CTA​

指标条下方有一行主操作按钮，覆盖高频场景：

按钮行为
快捷开发（primary）把"一键开发"提示词填入 AI Dock，由 Agent 根据当前设备规划开发任务
终端切到 远程终端 tab，默认打开第一个 SSH 标签
OpenClaw切到 OpenClaw tab，查看板端 Agent 状态
设备体检把"设备体检"提示词填入 AI Dock，Agent 自动跑一轮诊断

能力徽章（Capability Badges）​

主 CTA 下方是一排"能力徽章"，展示工作区模块的安装与运行状态，点击直接跳到对应 tab：

徽章对应 tab状态含义
开发环境远程终端绿点 = 就绪；灰点 = 未启动；红点 = 未安装
IDE远程 IDE同上（code-server 安装并运行时显示就绪）
远程桌面远程桌面同上（NoVNC 服务运行时显示就绪）

徽章颜色反映板端服务的真实状态——绿点代表点进去就能用；灰点代表服务已装但未起（点击进入后 Studio 会引导启动）；红点代表尚未安装（点击进入后 Studio 会引导安装）。

离线诊断提示​

指标条下方如果显示橙色小条（如"设备密码不正确或缺失，请在设备管理中重新连接"），说明最新一轮探测失败。这条提示会区分以下几种情况：

  • 认证失败（密码错误、SSH 密钥缺失）
  • 响应超时（板掉线或网络拥塞）
  • 连接中断（网线/Wi-Fi 物理断开）
  • 其他采集失败

如果整体网络正常、但提示"未联网"，AI 对话和软件安装依赖的功能会暂时不可用，但工作台基础信息仍会读取缓存。
